# Сердца, скреплённые скотчем (отзыв)

![HeartsTapedTogether](https://gretmn102.github.io/HeartsTapedTogetherReview/HeartsTapedTogether.png)

Кто такой пионерскаут? Что сильнее: любовь или обстоятельства? О чем может поведать бродячий философ, оставшийся без крова над головой? Как доставить букет цветов возлюбленной с помощью камня и скотча? На эти и другие вопросы вы узнаете ответ в игре [Сердца, скреплённые скотчем](https://quest-book.ru/online/view/game18149/) автора Pete Pr!

В аннотации указано, что игра — продолжение [Спасение рядовой ели](https://quest-book.ru/online/view/game18149/). Но читается как самостоятельное произведение.
